Stable Diffusion Slack Bot

Giphy, but each image has never been seen before.

This project built a stable diffusion Slack bot using a spare GPU in the York computer vision lab and Slack API. The project was completed over an evening and deployed internally on our slack workspace. The above image was generated using the prompt: “A 3D model of a ghost eating a burger”. Simply ‘@’ the stablediff bot with a prompt, and it will return six samples of never before seen generated images from the text.
